About This Tool

This page demonstrates embedding an AI assistant into a vanilla JavaScript site using a remote script.

Key Features

  • Easy integration with minimal setup
  • Supports dynamic configuration
  • Secure API key usage

How It Works

The assistant script is loaded asynchronously and initialized with your API key and assistant ID. Once loaded, it appears as a button in the bottom-right corner of the screen.
